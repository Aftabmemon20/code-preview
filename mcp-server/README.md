# MCP Server with Firebase OAuth and Multi-language Code Preview

## Setup
1. Clone repo
2. Setup backend:
   ```
   cd backend
   cp .env.example .env
   npm install
   npm start
   ```
3. Setup frontend:
   ```
   cd frontend
   cp .env.example .env
   npm install
   npm run dev
   ```
4. Deploy as needed (Firebase Hosting for frontend, Render/Heroku for backend)
